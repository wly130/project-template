### 下载依赖包

```
npm install
```

### 运行项目

```
npm run start
```

### 目录/文件

| 目录/文件                 | 作用            |
| ------------------------- | --------------- |
| bin                       | 启动文件        |
| config                    | 配置目录        |
| config/error-config.js    | 错误配置        |
| config/log-config.js      | 日志配置        |
| config/mysql-config.js    | mysql数据库配置 |
| config/token-config.js    | token配置       |
| config/validate-config.js | 验证器配置      |
| conttroller               | 控制层          |
| log                       | 日志文件目录    |
| models                    | 数据表目录      |
| routes                    | 路由表目录      |
| routes/index.js           | 路由表入口文件  |
| validator                 | 验证器目录      |
| app.js                    | 项目入口文件    |
| package.json              | 依赖配置文件    |
