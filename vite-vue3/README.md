- 下载依赖包

	```
	npm install
	```

	### 运行项目

	```
	npm run start
	```

	### 目录/文件

	| 目录/文件          | 作用         |
	| ------------------ | ------------ |
	| public             | 公共文件     |
	| src/api            | 接口配置目录 |
	| src/api/api.js     | 接口配置文件 |
	| src/api/requset.js | Axios封装    |
	| src/assets         | 静态资源文件 |
	| src/components     | 组件目录     |
	| src/pages          | 页面目录     |
	| src/router         | 路由配置目录 |
	| src/utils          | 工具目录     |
	| src/App.vue        | 入口页面     |
	| src/main.js        | 入口文件     |
	| index.html         | 项目入口文件 |
	| package.json       | 依赖配置文件 |
	| vite.config.js     | 项目配置文件 |
